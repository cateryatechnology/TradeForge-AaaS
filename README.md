## 🚀 Quick Start

```bash
# Clone & Setup
git clone https://github.com/cateryatechnology/TradeForge-AaaS.git
cd TradeForge-AaaS
cp .env.example .env

# Edit .env with your API keys

# Start with Docker
docker-compose -f docker/docker-compose.yml up -d

# Access: http://localhost:8501
```
